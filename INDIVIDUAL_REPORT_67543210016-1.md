## ขอบเขตงานที่รับผิดชอบ

## สิ่งที่ได้ดำเนินการด้วยตนเอง
อธิบายสิ่งที่ลงมือพัฒนาด้วยตนเองโดยสรุป เช่น
- ทดสอบ API เช่น Login และ Task API
- ตรวจสอบการทำงานของ JWT Authentication
- ทดสอบการใช้งานระบบผ่าน HTTPS
- จัดเตรียม screenshots สำหรับการส่งงาน
- ช่วยจัดทำ README และเอกสารประกอบโปรเจกต์

## ปัญหาที่พบและวิธีการแก้ไข
พบปัญหาไม่สามารถ login ด้วย seed users ได้ เนื่องจาก password ในฐานข้อมูลยังไม่ได้เข้ารหัส bcrypt จึงต้องสร้าง bcrypt hash ใหม่และแก้ไขในไฟล์ db/init.sql ก่อนรันระบบ

## สิ่งที่ได้เรียนรู้จากงานนี้
ได้เรียนรู้แนวคิดของ Microservices Architecture, การใช้ JWT Authentication, การทำงานของ API Gateway (Nginx) และการใช้ Docker Compose เพื่อรวมหลาย services ให้ทำงานร่วมกัน
